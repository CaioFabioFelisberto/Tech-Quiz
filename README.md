#titulo
Quiz Tech

##descrição/instructions
Para o quiz funcionar, certifique-se de que o MongoDB esteja rodando localmente e que a coleção "perguntas" esteja populada com as questões do quiz. Certifique-se também de que as variáveis de ambiente no arquivo .env estejam configuradas corretamente. Além disso, instale as dependências necessárias usando pip install flask pymongo python-dotenv.

O quiz é acessível através do navegador, e as perguntas são apresentadas uma a uma. O usuário pode responder e receber feedback imediato sobre suas respostas, além de um resumo final ao completar o quiz. Para responder o quiz, acesse http://localhost:5000/1 no seu navegador.

#title
Quiz Tech

#description/instructions
For the quiz to work, make sure MongoDB is running locally and that the "perguntas" collection is populated with the quiz questions. Also, ensure that the environment variables in the .env file are set correctly. Additionally, install the necessary dependencies using pip install flask pymongo python-dotenv.

The quiz is accessible through the browser, and questions are presented one by one. The user can answer and receive immediate feedback on their answers, as well as a final summary upon completing the quiz. To access the quiz, go to http://localhost:5000/1 in your browser.
